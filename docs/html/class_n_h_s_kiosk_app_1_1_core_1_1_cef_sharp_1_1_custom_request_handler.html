protected

Called before browser navigation. If the navigation is allowed IChromiumWebBrowserBase.FrameLoadStart and IChromiumWebBrowserBase.FrameLoadEnd will be called. If the navigation is canceled IChromiumWebBrowserBase.LoadError will be called with an ErrorCode value of CefErrorCode.Aborted.

Parameters
chromiumWebBrowserthe ChromiumWebBrowser control.
browserthe browser object.
frameThe frame the request is coming from.
requestthe request object - cannot be modified in this callback.
userGestureThe value will be true if the browser navigated via explicit user gesture (e.g. clicking a link) or false if it navigated automatically (e.g. via the DomContentLoaded event).
isRedirecthas the request been redirected.
Returns
Return true to cancel the navigation or false to allow the navigation to proceed.
